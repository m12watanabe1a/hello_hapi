# hello_hapi
Say Hello to Hapi

## Requiremnts
 - tensorflow >= 2.0.0
 - opencv-python >= 4.1.0
 
### Install requiremnts
 ```bash
  $ python3 -m pip install -r requirements.txt
 ```

# Run
```bash
 $ ./hello_hapi
```
